Freight is a full-stack logistics management platform built using **MERN Stack + Vite**.  
It supports **real-time shipment tracking**, **global map visualization**, **automated login alerts**, and **email workflows**.

## Features
- Secure JWT authentication  
- Shipment creation & management  
- Real-time GPS tracking with Mapbox  
- Automated login notifications (Resend Email API)  
- REST API built with Node.js & Express  
- Backend deployed on Render  
- Frontend deployed on Vercel
